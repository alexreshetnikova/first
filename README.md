# **NeuroStartUp**

## Начало работы
Для получения доступа к проекту оставьте заявку на [нашем официальном сайте.](https://www.neurostartupchallenge.org/)

### Prerequisites
Для работы с проектом на вашем ПК должны быть установлены:

* Git
* Браузер Chrome
